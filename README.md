Lecture Notes

Overall the Lectures were based on learning more about Game Design Patterns, specifically about Singletons and Factories.

A Singleton is essentially a Game Manager, which can help with backend scripts, global settings, level management, and save progression.

A Factory helps create objects belonging to a superclass or larger group of things. Something like an enemy spawner would be considered a Factory.
